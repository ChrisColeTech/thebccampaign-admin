Administer Comments

Empowering Dialogue, Safeguarding Security

This page is designed for campaign administrators to manage comments and ensure a secure and engaging environment for online discussions.

Features:

  • Security Highly secure platform to protect user data and maintain confidentiality.
  • View Comments Access and manage published comments. The page includes a 'Donate' button to facilitate campaign contributions, sponsorships, and endorsements.
  • Export Comments Seamlessly export all approved and unapproved comments to a flat XLSX file for comprehensive analysis. The comments are unordered, allowing flexibility to apply filtering, grouping, sorting, and more using Excel's powerful functionalities.

Please Note:

  • Permanent approvals Once a comment is approved, it cannot be undone unless overridden by a Database Administrator (DBA).
  • Permanent deletes Deleted comments are permanently removed from the system.
  • Changes Impacting Widget Modifications made on the 'View Comments' page will reflect on the public-facing Widget, ensuring consistent messaging and user experience.
